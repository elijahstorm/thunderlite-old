Information about scripting

But don't worry, you don't need to be a programmer or know game logic to get started. It's as easy as English!

Conditions

In order to start reading code, you have to start it in a Conditional.

Conditionals start with < condition "value" >

When the condition is met, the script within the condtion tag will be ran.

To end a conditional, just run the < / condition > end tag. End tag with the "/" at the start.


Example

...
<start>

talk Jeff: "I'm going to highlight some tiles for you."

move: 16,15
hl: 9, 11
hl: 9, 14
hl: 11, 14

wait: 2
talk Jeff: "Alright, there you go. I'll take the highlights away now."

unhl: 9, 11
unhl: 9, 14

</start>

All Actions

talk Name : "Text1","Text2",...,"TextN"
 (each text must be in quotes)
wait : Time (in seconds)
move : Tile x, y
 (0, 0 starts in the top, left. and moves down and to the right. x in horizonal. y is vertial. comma to seperate)

set player Number : Name

hl : Tile x, y
unhl : Tile x, y
 (if you don't unhighlight, it will stay highlighted until the end of the game or if it's changed via terrain action)

terrain : "Terrain Type Name", Tile x, y

 (Terrain name needs to be in quotes. If a type happens, code will ignore it. Check your spelling if it doesn't work.)

hurt unit : Tile x, y Change unit health
add unit : Tile x, y Adds new unit to game
kill unit : Tile x, y Removes unit from game

hurt city : Tile x, y Change city capture rate
add city : Tile x, y Adds new city to game
kill city : Tile x, y Removes city from game
