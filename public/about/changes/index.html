MULTIPLAYER - 4/15/2020

Want to play live MULTIPLAYER? Want to create and host original maps?

Both those features are already included in the game! Feel free to test it out by inviting your friends to play in a Multiplayer Custom Map game! More info below.

Try it! https://thunderlite.herokuapp.com/

Want to create a map? Just click Map Editor on the dropdown menu to the left to begin creating! Once you're finished, test it. If you complete the map (proving it's possible), you can UPLOAD it so everyone else can play!

Want to play a game? Go to MULTIPLAYER and host an existing game, or search for one of your own!

Want to play with a friend? Invite them to join with the link above. New features include fixing all those server errors that use to pop up. Now there's no more CONNECTION issues. If a user loses connection Mid-Game, the server will allow them to rejoin within 30seconds to get right back into the game!

Think it's too good to be true? Test it out! Invite a friend to play with you, then after a turn or two, have them close the tab. Then reopen it and log back in, your game will automatically resume! Just don't wait more than 30seconds, or the sever will assume they aren't intending to reconnect and end the game.



Play around and relive some old memories! Invite your friends and maybe we can regain the old community :)
